# Change Log

All notable changes to the "json-fragments" extension will be documented in this file.

## [0.0.2] - 2026-05-01

- Added static JSON fragment preview tabs with source navigation support.
- Added dynamic preview for the fragment at the active editor selection.
- Added configurable file include and exclude filters.
- Added hover, scanner, preview, and tracker configuration settings.
- Improved visible-range tracking and test coverage for preview and configuration behavior.

## [0.0.1] - 2026-05-01

- Initial JSON fragment detection for objects and arrays embedded in text files.
- Formatted hover preview for detected JSON fragments.
- Inline highlighting for scanned visible editor ranges.
- Manual active-editor scanning and temporary focused-file tracking commands.
- Domain-scoped scanner and tracker settings.
